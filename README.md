# VolumeHandControl
Python project to control audio volume with hand gestures

Running the code:
* `python venv .venv`
* `source .venv/bin/activate`
* `python -m pip install -r requirements.txt`
* `python VolumeHandControl[Alsa/Pipewire].py`
